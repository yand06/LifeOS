# Problem

Sebagian besar aplikasi ToDo List hanya fokus pada daftar pekerjaan.

Masalahnya:

- tidak ada feedback
- tidak ada perkembangan visual
- tidak ada motivasi
- checklist besok kembali kosong

Akibatnya pengguna berhenti menggunakan aplikasi.

LifeOS mencoba mengubah checklist menjadi permainan kehidupan.
