# Vision

Membuat aplikasi yang membuat pengguna merasa ingin membuka aplikasi setiap hari.

Bukan karena ada notifikasi.

Tetapi karena ingin melihat perkembangan dirinya sendiri.

Target utama:

- meningkatkan disiplin
- mengurangi bad habit
- membangun good habit
- membuat self improvement menjadi visual
- memberikan insight berbasis data
