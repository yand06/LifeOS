# LifeOS

LifeOS adalah aplikasi personal productivity yang menggabungkan checklist, habit tracker, analytics, dan gamification.

Tujuan utama bukan sekadar menyelesaikan tugas, tetapi membantu pengguna membangun kebiasaan baik melalui visualisasi perkembangan diri.

Konsep utama:

> One Day = One Match

Setiap hari adalah pertandingan.

Setiap aktivitas akan menambah atau mengurangi skor harian.

Semua aktivitas dikategorikan menjadi:

- Good Habit
- Natural Activity
- Bad Habit

Semua aktivitas akan direkam untuk menghasilkan insight mengenai perkembangan pengguna.
